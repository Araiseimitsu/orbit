このアプリの使用方法

クイックスタート

  1. Dashboard から新規作成（または Import で YAML 取り込み）
  2. ワークフロー名 / トリガー（手動・スケジュール）/ Active を設定
  3. 左のアクションからノード追加、右の詳細でパラメータ設定
  4. 実行順は上から下。ドラッグで並び替え可能
  5. 保存すると workflows/ に YAML が自動生成
  6. Dashboard で Run / Schedule 実行、Runs で履歴・結果・出力を確認

AI フロー自動構築

  • 文章からステップ構成を生成（置き換え / 追加）
  • ファイルパス・ルーム番号・シート名・範囲・条件・出力キーなど細かく書くほど精度が上がります
  • Web検索は必要時のみ ON（既定は OFF）
  • 生成後はノードをクリックして詳細調整できます

条件分岐と参照

「条件を使う」で step / field / match / equals を指定。比較は前後空白除去＋大文字小文字を無視します。 テンプレートは {{ '{{ step_id.key }}' }} で前ステップ結果を参照できます。

主要アクションの例

  • AI 生成: プロンプトを渡して要約・分類・生成（Web検索対応）
  • AI 判定: yes/no で判定、条件分岐と組み合わせ可能
  • ARAICHAT 送信: room_id / text / files を指定して送信
  • Excel: 読み込み / 書き込み / 追記 / シート一覧
  • Google Sheets: 読み込み / 書き込み / 追記 / シート一覧
  • Notion: データベース検索 / ページ作成 / ページ更新
  • ファイル: 読み書き / コピー / 移動 / 削除 / リネーム
  • 判定: 完全一致 / 部分一致 / 正規表現 / 数値比較
  • サブワークフロー: 他のワークフローを呼び出し
  • Log: 任意メッセージを実行ログに記録

設定とキー類

  • Gemini API キー: secrets/gemini_api_key.txt または GEMINI_API_KEY
  • ARAICHAT API キー: secrets/araichat_api_key.txt または ARAICHAT_API_KEY
  • Google Sheets: secrets/google_service_account.json または GOOGLE_APPLICATION_CREDENTIALS
  • Notion API キー: secrets/notion_api_key.txt または NOTION_API_KEY

環境変数が優先されます（.env ファイルで設定可能）
