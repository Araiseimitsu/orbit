AI Agent Core Rules
言語
全ての回答・コメントは日本語で記述すること
プロジェクト構成
プロジェクトルート直下に src/ 階層を作成する
実装は必ず src/ 配下に配置する
ドキュメント管理
プロジェクト直下に .docs/ ディレクトリを作成する
重要な変更履歴は .docs/update.md にのみ記録する
些細な変更・試行錯誤は記載しない
Python 環境
Python Install Manager を使用する
実行形式は以下に限定する
py -3.xx script_name.py
コード品質（厳守）
1ファイルあたり 1000行以内
重複コード・冗長構造は禁止
単一責任原則を守り、機能単位でモジュール分割する
共通処理は必ず別ファイルに分離する
フロントエンド（厳守）
• HTML / CSS / JavaScript は必ず別ファイルに分離する
• インラインスタイル・インラインスクリプトは禁止
• 拡張子ごとに明確に管理する
.html
.css
.js
データベース操作（最重要）
削除・初期化・リセット処理は必ず事前にユーザー承認を得る
実行前に影響範囲を明示する
承認なしでの自動実行は禁止
デバッグ
詳細なログ出力を必ず実装する
エラー発生時はデバッグモードで原因特定を優先する (edited)
